# Agent Rules

## Purpose

Single consolidated rule file directing agents to their specific documentation and required skills.

## For Main Agent

**Read your agent documentation**: `.agents/agents/main-agent.md`

## For Sub-Agents

**Read your agent documentation**: Provided by Main Agent when spawned

## Core Principle

**Every agent MUST read their documentation file which specifies which skills and rules to load.**

Agent documentation files are located in `.agents/agents/[agent-name].md`
