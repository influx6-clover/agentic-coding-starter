# Skills Usage (For Sub-Agents)

## Purpose

Concise guide for sub-agents using existing skills.

## Rule

**Read this skill**: `.agents/skills/skills-management/skill.md`

## Summary

**Before using skill:**
1. Find in `.agents/skills/` (scan frontmatter)
2. Verify `approved: Yes`
3. Read complete `skill.md`
4. Check Usage Type (TEMPLATE/EXECUTABLE/EDUCATIONAL)

**Usage Type Rules:**
- **TEMPLATE**: Copy files to project, customize, import from project
- **EXECUTABLE**: Run scripts from skills/, use output
- **EDUCATIONAL**: Install external libs, write fresh code

**Golden Rule**: Never import from `.agents/skills/` in project code.
